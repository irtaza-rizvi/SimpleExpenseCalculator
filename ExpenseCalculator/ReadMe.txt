**********Usage**********

Step1:Compile and run Expenses.java

Step2:Enter Month which will create a text file of the Month's name in the same folder

Step3:Enter the Amount and Item for each expense in the following way:-
AmountItemAmountItemAmountItemAmountItemAmountItem

Example:-
10.5food0.2print12bus20food6bus5food40phonebill500houserent30food

Note:Expenses with the item name 'food', 'home', 'print', 'film', 'bus' and 'metro' will be added together.
